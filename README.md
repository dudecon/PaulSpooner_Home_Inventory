\# Home Inventory Vault



\*\*Structure\*\*

\- `Locations/` = fixed spaces (may have utilities)

\- `Containers/` = movable storage

\- `Items/` = real owned stuff

\- `Wishlist-Locations/` = idealized future workspaces (blueprints)

\- `Wishlist-Containers/` \& `Wishlist-Items/` = planned purchases

\- `Projects/` = actionable diffs (maintenance or big upgrades)
